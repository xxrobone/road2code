Where to start?

  • What I got recommended to try out and did, was this course at FreeCodeCamp
    Responsive webdesign - freecodecamp
  • I checked some tutorials on Youtube started with Brad Traversys HTML tutorial
    HTML - Tutorial
  • And also a CSS tutorial by Traversy
    CSS - Tutorial
  • I also started reading the ELOQUENT JAVASCRIPT BOOK, its also available online
    Eloquent Javascript
  • In combination with everythings I went on and bought the Developer Bootcamp course at UDEMY by Colt Steele (its also updated)
    The Web-developer bootcamp - UDEMY

...Alongside with taking a bunch of tutorials, reading online and studying from different sources I also tried to build small projects and also later continued on to try out frameworks, in my case I chose React and Next. Also tried out Wordpress as a CMS just to learn it a bit
